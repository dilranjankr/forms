HELP & RULES
Invoice — how to use
New LDP / LCP / Change Order: pick Section Type, Column Name, Date, add line items → Save to Workbook.
Payment Request: pick "Payment Request" — lines auto-load from PR#TBB → edit amounts + set Column Name (e.g. PR#1) → Save to Workbook (Vendor Tracking is linked automatically).
Finish: Run Invoice Generate — refreshes PR#TBB and makes the PR invoice tabs.
Purchase Orders — how to use
Add POs: choose Source contract, fill PO rows → Run Update Po.
Move POs: Run Load Po to Move → tick rows → choose Target → Run Move Po.
Adjustment: select Source/PO, enter Contract Adjust + Description → Run Contract Adjust.
⚠ Do NOT change in the workbook
• Sheet names (Invoice Worksheet, Vendor Tracking, PR#TBB, PO Input Form, Payments) — don't rename.
• The bold LDP / LCP marker rows in column A — LDP vs LCP is decided by these (not by the contract name).
• Header text: Total To date, Paid to Date, Payment Request, LDP Total, Total, PR#TBB, Grand - TOTALS, SUB - TOTALS - LDP/LCP — keep the exact spelling.
• Don't delete the PR#TBB / CO#TBB columns — new columns are inserted before them.
• Every Payment Request makes a NEW column. PR#1, PR#2… tabs are frozen snapshots; only PR#TBB stays live.
