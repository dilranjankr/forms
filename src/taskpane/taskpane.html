▼ 1. Quick Start
New LDP / LCP / Change Order
  1. Pick the Section Type.
  2. Enter a Column Name (e.g. Kiser LCP Phase 1, CO#2).
  3. Pick a Date.
  4. Fill Vendor Tracking (only for a brand-new contract).
  5. Add line items (no amount = bold sub-header).
  6. Click Save to Workbook.
Payment Request
  1. Select Payment Request — the form auto-loads PR#TBB lines.
  2. Edit the amounts that should appear on this PR.
  3. Set the Column Name (e.g. PR#3 — the latest PR name is shown above the field).
  4. Click Save to Workbook.
  5. Vendor Tracking is updated automatically.
Generate Invoice

When you're ready to issue invoices, click Run Invoice Generate. It refreshes PR#TBB and creates the frozen PR#1, PR#2… invoice tabs.

▶ 3. Glossary
LDPLandscape Design and Permitting — the design phase of a project.
LCPLandscape Construction Phase — the build phase.
COChange Order — additional scope added after the original contract.
PRPayment Request — an invoice issued to the client.
PR#TBBPayment Request — To Be Billed. A live, rolling column that holds every line waiting to be invoiced.
POPurchase Order — issued to a vendor / sub-contractor.
ESTEstimate number from a vendor.
VTVendor Tracking sheet — per-contract vendor + PO breakdown.

▶ 5. Workbook Rules — Do NOT change
  • Do not rename the sheets — Invoice Worksheet, Vendor Tracking, PR#TBB, PO Input Form, Payments.
  • Do not delete the bold LDP / LCP marker rows in column A — LDP vs LCP is decided by these rows, not by the contract name.
  • Keep the exact spelling of these header strings: Total To date, Paid to Date, Payment Request, LDP Total, Total, PR#TBB, Grand - TOTALS, SUB - TOTALS - LDP/LCP.
  • Do not delete the PR#TBB or CO#TBB columns — new columns are always inserted just before them.
  • Every Payment Request creates a new column. The PR#1, PR#2 tabs are frozen snapshots; only PR#TBB stays live.
  • Do not edit row 5/6 of the new PR columns in Vendor Tracking by hand — they are live formulas linking to the Invoice Worksheet.
▶ 6. Troubleshooting
Panel stuck on "Loading…"
  • Press Ctrl + Shift + R to hard-refresh the browser.
  • Close the Excel tab completely and reopen the workbook.
Icons look black or text isn't styled

The Office cache is stale. Press Windows + R, paste this path, delete everything inside, then reopen Excel:

%LOCALAPPDATA%\Microsoft\Office\16.0\Wef
Date is not saving in the Invoice Worksheet

Make sure the Date field is filled before you click Save to Workbook. The date goes into row 4 of the new column.

Add-in name still shows the old label

Office caches the manifest. Sign out and sign in again from File → Account, or wait up to 24 hours for the auto-refresh.
